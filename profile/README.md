**Developer tools for Delphi.** Cloud connectivity, HTTP testing, and more.

Appercept is a software company founded by Richard Hatherall, a developer who
has been working with Delphi since version 1. We build libraries and tools for
the Delphi community, with a focus on bringing AWS support to a language that
was left behind by the cloud.
